Coursera: Google UX Design, September 2022 – February 2023

Created 3 projects in Figma using design sprints to make a primary user flow for a mobile app, responsive website, and a project involving a mobile app and website. I began research by conducting user interviews, empathy mapping, personas, user stories, and user journey maps. The ideation began by sketching paper wireframes, low fidelity prototypes, creating a research plan, and usability studies. Next, insights and themes were generated to create high fidelity prototypes, information architecture, and site maps for each project. Finally, I created design systems for each project.

02. UX Projects

Local Bakery Ordering App

Village Bread is a local bakery in my community that sells artisan bread and baked foods. This project solved an important issue which helped busy users who can't order from the bakery in person. With the online ordering system created for this projects, users will be able to place orders and pick up their products at a more convenient time.

View Project

Indie Band UX Design

Daydream Time Machine is a local indie band that doesn't have a centralized website. This project addressed the issue and established an engaging online platform that enhanced the fan experience by facilitating the discovery of shows, browsing albums, and purchasing band merchandise.

View Project

User to User Book Rental App

The RoseWater app is a unique app design that solved the issue with finding affordable book rentals. The app allows users to lend their books to other users directly rather than having to purchase or rent books from more expensive services.
